# Linux filesystem

`/etc` - Systems version of `~/.config`. (equivalent of registry)

`/usr` - "Unix System Resources." Contains system resources. (Program Files / Program Files x86 / system32)

`/usr/share` Shared resources (used by multiple apps)

`/usr/bin` / `/bin` - `/bin` links to `/usr/bin` in most cases. Contains system executables. (system32 / C:\Windows)

`/usr/local` Usually, custom resources are placed here. (%appdata%\local except machine-wide)
